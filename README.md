# React Web Demo Showcase

A lightweight React-based web application demo focused on component composition, state-driven UI, and a clean dashboard-style experience.

## Overview

This project presents a small task board interface built with React and Vite. It is designed as a simple showcase of how a modern React application can be structured around reusable components, local state management, and mock data.

## Features

- Search tasks by title or summary
- Filter tasks by status
- Browse a responsive task card list
- View task details in a dedicated side panel
- Clean mock-data-driven UI with no backend dependency

## Tech Stack

- React
- Vite
- Modern CSS

## Project Structure

```text
src
  components
    TaskDetail.jsx
    TaskList.jsx
    Toolbar.jsx
  App.jsx
  data.js
  main.jsx
  styles.css
```

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Demo Focus

This demo is intended to showcase:

- React component-based architecture
- State lifting and prop-driven communication
- Filtering and selection logic in a small UI
- A practical starter structure for future expansion
